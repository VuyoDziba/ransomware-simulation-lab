# Ransomware Simulation Lab

## Overview

This project demonstrates a **simulated ransomware payload** created for educational purposes in a controlled Kali Linux environment.

The goal is to understand how ransomware operates including:

* File creation
* Text scrambling
* Encryption using AES-256
* Fake ransomware popup
* Browser redirection
* Automatic logout

This project was created as part of **Ethical Hacking coursework**.

---

## Technologies Used

* Bash Scripting
* OpenSSL
* Zenity
* Kali Linux
* AES-256 Encryption

---

## Features

✔ File creation with dummy data
✔ ROT13 reversible scrambling
✔ AES-256 encryption using OpenSSL
✔ Fake ransomware popup notification
✔ Automatic browser redirect
✔ Auto logout after execution
